### 📝 Not Defteri

Modern, hızlı ve hafif bir kişisel not defteri uygulaması. **Rust** ile yazıldı, **egui** ile arayüzü oluşturuldu.

## ✨ Özellikler

- **Markdown desteği** — Notlarınızı Markdown formatında yazın, tarayıcıda önizleyin
- **Kategorilendirme** — Notlarınızı kategorilere ayırın
- **Etiketleme** — Notlarınıza etiket ekleyip filtreleyin
- **Arama** — Tüm notlarda anında arama
- **Filtreleme** — Kategori ve etikete göre notları filtreleyin
- **Dışa aktarma** — Notları Markdown dosyasına aktarın
- **Modern arayüz** — egui ile hızlı ve temiz bir kullanıcı deneyimi
- **Otomatik `.desktop` entegrasyonu** — İlk çalıştırmada sistem menüsüne eklenir

## 📖 Kullanım

Uygulamayı çalıştırın:

note-app

- **Sol panel**: Not listesi, arama çubuğu, kategori/etiket filtreleri
- **Sağ panel**: Başlık, kategori, etiket girişi ve Markdown editörü
- **Kaydet**: Değişiklikleri kaydeder
- **Sil**: Seçili notu siler
- **Önizle**: Markdown içeriğini HTML'e çevirip tarayıcıda açar
- **+ Yeni**: Yeni not oluşturur

## 🗂 Depolama

Notlar `~/.local/share/note-app/notes.json` dosyasında JSON formatında saklanır.
